Application CovidTracker

Utilisation de l' Api Covid19, une API sur les données de la maladie du CoronaVirus.

Application conçue pour savoir en temps réel, le nombre de cas de malades du Covid
(Nombre : de nouveaux cas, total, de morts, répartition par pays, par région,...)
Ces projets utilisent NodeJs, Vue3, Vite, Axios, et Vuex pour le state.

Technologies : HTML5, CSS3, JavaScript, Node.js, Vue3.
Dependances : Axios, Moment, Core-js, Tailwind, Vue-Router, VueX.
